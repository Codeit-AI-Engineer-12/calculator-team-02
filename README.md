# 계산기
: Python으로 구현한 기본 계산기 프로젝트

## 지원 기능

|        Name         |               Description                   |
|:--------------------:|:------------------------------------------:|
|      ```ops.add.add(a, b)```      |   두 자연수의 합을 반환합니다   |
| ```ops.subract.subtract(a, b)```  | a를 b로 뺍니다.<br>~~<i>(더 이상의 자세한 설명은 생략한다.)</i>~~ |
| ```ops.multiply.multiply(a, b)``` |   두 자연수를 곱한 결과를 반환한다.   |
|   ```ops.divide.divide(a, b)```   |                               |
|     ```ops.power.power(a, b)```      |   밑수(a)를 지수(b)만큼 거듭제곱한 값을 반환합니다  |
