# 계산기
: Python으로 구현한 기본 계산기 프로젝트

## 지원 기능

|                   Name                    |          Description           |
|:-----------------------------------------:|:------------------------------:|
|          ```ops.add.add(a, b)```          |      자연수인 a와 b의 합을 연산합니다.      |
|     ```ops.subract.subtract(a, b)```      |        a와 b의 빼기를 연산합니다.        |
|     ```ops.multiply.multiply(a, b)```     |        a와 b의 곱셈을 연산합니다.        |
|       ```ops.divide.divide(a, b)```       |       a와 b의 나눗셈을 연산합니다.        |
|        ```ops.power.power(a, b)```        | 밑수(a)를 지수(b)만큼 거듭제곱한 값을 연산합니다. |
|      ```ops.minimum.minimum(a, b)```      |     두 자연수 중 더 작은 값을 반환합니다.     |
|         ```ops.root.root(a, b)```         |       a를 b로 거듭제곱근 반환합니다.       |
|          ```ops.avg.avg(a, b)```          |       a와 b의 평균값을 연산합니다.        |
|          ```ops.max.max(a, b)```          |       a와 b중 더 큰 값을 연산한다.       |
| ```ops.floor_divide.floor_divide(a, b)``` |        정수 나눗셈의 몫을 연산한다.        |
|          ```ops.mod.mod(a, b)```          |        a를 b로 나머지 연산합니다.        |
|          ```ops.fact.fact(x)```           |         x를 팩토리얼 연산합니다.         |
